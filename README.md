# Тестовое задание на позицию React Native Developer

#### Разработчик - _Алексеев Ростислав_

## Декомпозиция проекта (базовая)

### 1/2 День

1. Провести базовую настройку проекта
   - Создать скелет FSD архитектуры
2. Создать начальный скелет приложения
   - Вёрстка главного экрана (AppBar, EmployeeCard, SortPopup)
   - Вёрстка экрана пользователя
   - Вёрстка экрана ошибки
3. Начать реализацию основных фич
   - Интеграция и тестирование API

### 2 День

1. Реализация основных фич
   - Загрузка данных работников
   - Поиск и фильтрация пользователей
   - Переход на страницу работника
   - Сортировка пользователей
2. Рефакторинг и тестирование
3. Начать реализацию дополнительных заданий
   - Real-time отслеживание состояния сети устройства
   - Сохранение состояния фильтрации и поиска

### 2/3 День

1. Закончить основные фичи
2. Закончить реализацию дополнительных заданий
   - Кэширование результатов запроса по различным параметрам
   - Реализовать темную тему для приложения
   - Добавить локализацию
3. Протестировать приложение и все фичи
   - Добавить иконки
